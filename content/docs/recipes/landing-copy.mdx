## Steps

1. **Claude Project.** Run [Prompt 4 Landing page spec](/docs/prompts/landing-page-spec). Output is a structured spec with hero, problem, offer, how it works, social proof slot, CTA.
2. **ChatGPT free.** Paste the spec. Ask for a full-page draft with the exact wording for each section. GPT-5.3 is stronger than GPT-5.3-mini for this, so spend your first 10 messages of the 5-hour window here.
3. **ChatGPT free.** Same window. Run [Prompt 9 Strip AI smell](/docs/prompts/strip-ai-smell) on the draft. Spend the GPT-5.3 messages before they drop to mini.
4. **Claude Project.** Final read. If it sounds human in 3 seconds, ship.
