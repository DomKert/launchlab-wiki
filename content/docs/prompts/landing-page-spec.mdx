## Why this works

One prompt, one structured output you paste into any builder (Bolt, v0, Carrd, Framer, raw HTML). You show up with a 4-box, you leave with a spec. No bouncing between five "help me write my hero section" chats. Run once Friday evening to ship a rough WIP, again Saturday morning to polish.

## Prompt

```txt
Turn my 4-box into a landing page spec I can build in [Bolt / v0 / Lovable / Artifacts / Carrd / Framer / Webflow / raw HTML, pick one and say it].

Output these sections in order, with copy for each:
1. Hero: headline (6 to 8 words), subhead (one sentence), primary CTA button text (2 to 4 words)
2. The problem (2 short paragraphs, what breaks today without me)
3. What I actually offer (3 bullet points, each starts with a verb)
4. How it works (3 numbered steps, one line each)
5. Social proof slot (what would go here when I have it, even if it's empty today)
6. Final CTA block (one line + button text)

Voice rule. Sound like a founder texting a friend, not a brochure. No corporate verbs, no colons-then-lists, no "revolutionary" / "innovative" / "unlock." If you use a word I wouldn't say out loud, rewrite it.

Also give me 3 visual hints for the hero (mood, 1 to 2 color ideas, one photo style or none). Keep the spec under 500 words total.
```
