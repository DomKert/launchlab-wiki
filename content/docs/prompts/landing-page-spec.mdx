## Why this works

One prompt, one structured output you paste into any builder (Bolt, v0, Carrd, Framer, raw HTML). You show up with a 4-box, you leave with a spec. No bouncing between five "help me write my hero section" chats. Run once Friday evening to ship a rough WIP, again Saturday morning to polish.

## Why ChatGPT not Claude

This prompt drafts hero headlines, button text, and pitch copy. That's copy work, and ChatGPT writes punchier consumer copy by default than Claude (which leans measured and explanatory). Your Claude Project keeps its quota for planning. The Project chat already produced your customer.md, one-liner.md, 4-box.md and reviewed your research; let it sit while ChatGPT does the copy lift.

## Before you run this

You need `4-box.md` ready to paste. [Prompt 2 The 4-box](/docs/prompts/four-box) prints it as a code block at the end of its output. Copy that block into the placeholder below.

## Prompt

```txt
Here's my 4-box:

[PASTE 4-box.md content here]

Turn it into a landing page spec I can build in [Bolt / v0 / Lovable / Artifacts / Carrd / Framer / Webflow / raw HTML, pick one and say it].

Output these sections in order, with copy for each:
1. Hero: headline (6 to 8 words), subhead (one sentence), primary CTA button text (2 to 4 words)
2. The problem (2 short paragraphs, what breaks today without me)
3. What I actually offer (3 bullet points, each starts with a verb)
4. How it works (3 numbered steps, one line each)
5. Social proof slot (what would go here when I have it, even if it's empty today)
6. Final CTA block (one line + button text)

Voice rule. Sound like a founder texting a friend, not a brochure. No corporate verbs, no colons-then-lists, no marketing buzzwords. If you use a word I wouldn't say out loud at a coffee shop, rewrite it.

Also give me 3 visual hints for the hero (mood, 1 to 2 color ideas, one photo style or none). Keep the spec under 500 words total.
```

## Next: strip AI smell

Open a fresh ChatGPT chat (don't reuse the drafting one, fresh context catches more) and run [Prompt 9 Strip AI smell](/docs/prompts/strip-ai-smell) on the spec.
