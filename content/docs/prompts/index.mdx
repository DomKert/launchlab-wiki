Fifteen prompts for your 3-day weekend. Grouped by the same 4 modes as the [Tools](/docs/tools) loadout: Think, Research, Write, Build & Ship. Copy one, paste it into the primary tool on the card, fill in the bracketed parts, run it.

Two tags on each card. `planning` prompts go inside your Claude Project (the brain). Context accumulates across the weekend. `execution` prompts are portable. Paste into GPT, Perplexity, Gemini, Bolt, whatever the card says. Tier badge (S, A, B) shows priority if you only have time for a few.
