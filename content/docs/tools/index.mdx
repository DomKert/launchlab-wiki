## Multi-tool routing comes first

Claude drafts. ChatGPT stress-tests. Perplexity verifies. Gemini generates.

You will hit a rate limit on every tool this weekend. The workaround is routing, not paying. Plan in Claude (your Project), execute in the tool with the strongest free tier for that job, bring the output back to Claude for review.

## Three worked multi-tool workflows

Short recipes. Each one shows which tool does which step with the Back Room (a private invite-only European vintage brand) as the running example.

### Workflow 1. Day 1 customer research

<div className="my-3 flex items-center gap-2">
  <Badge variant="quest">DAY 1</Badge>
  <Badge variant="tier-s">S</Badge>
  <span className="text-sm">Perplexity scouts, Claude synthesizes.</span>
</div>

1. Perplexity free. Run one Pro Search on the niche. Example prompt: "5 to 8 currently-active competitors to a private invite-only European vintage drop brand for women 35 to 59. For each, one-line positioning and where they are weak." Save the output with citations.
2. Claude Project. Paste the Perplexity output. Ask Claude to synthesize into the 4-box (Customer, Problem, Value prop, Price). See [Prompt 2 The 4-box](/docs/prompts/four-box).
3. Claude reviews. Ask "which of these 4 boxes feels weakest and what would I need to know to tighten it."

Costs 1 of your 3 Perplexity Pro Searches per day. Saves Claude quota because Perplexity does the scraping.

### Workflow 2. Day 1 landing copy

<div className="my-3 flex items-center gap-2">
  <Badge variant="quest">DAY 1</Badge>
  <Badge variant="tier-a">A</Badge>
  <span className="text-sm">Claude briefs, ChatGPT drafts, Claude stress-tests.</span>
</div>

1. Claude Project. Run [Prompt 4 Landing page spec](/docs/prompts/landing-page-spec). Output is a structured spec with hero, problem, offer, how it works, social proof slot, CTA.
2. ChatGPT free. Paste the spec. Ask for a full-page draft with the exact wording for each section. GPT-5 is stronger than 4o-mini for this, so spend your first 10 messages of the 5-hour window here.
3. Claude Project. Paste the ChatGPT draft. Run [Prompt 9 Strip AI smell](/docs/prompts/strip-ai-smell) over it. Ship the cleaned version.

If ChatGPT output feels mushy mid-session, rotate to Gemini for the draft step.

### Workflow 3. Day 2 distribution visual

<div className="my-3 flex items-center gap-2">
  <Badge variant="quest">DAY 2</Badge>
  <Badge variant="tier-a">A</Badge>
  <span className="text-sm">Claude writes the image brief, Gemini generates, Claude polishes the caption.</span>
</div>

1. Claude Project. Ask for an image brief for a single-piece hero shot. "Neutral background, natural light, no model, mood quiet and archival, 4-to-5 ratio, 1 to 2 colors." Claude outputs a 2-sentence brief.
2. Gemini Nano Banana (or Ideogram). Paste the brief. Generate 2 to 3 options. Bring them back.
3. Claude Project. Ask Claude to critique the 3 options against the brief and pick one.
4. ChatGPT or Gemini. Draft an Instagram or LinkedIn caption for the chosen image in the voice from your voice guide.
5. Claude Project. Polish the caption. Cut AI filler, check for the banned patterns.

Dub.co short link on the final post so you can measure click-through.
