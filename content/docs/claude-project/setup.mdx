## 2. Paste the system prompt

Open [Core files](/docs/claude-project/core-files) in another tab. Copy the Meta guide block and paste it into the Project's Custom Instructions field. This is what makes the Project behave like your party leader instead of a generic chat.

<div className="my-3 rounded-md border border-fd-border bg-fd-card p-4 text-sm">
  <PersistentCheckbox id="setup-step-3" label="Pasted the Meta guide as the system prompt" />
</div>

## 3. Upload the context files

Four files from [Core files](/docs/claude-project/core-files) go into the Project knowledge. Click "Copy page" on the Core files page to grab the full bundle, or copy each block separately and save as .md or .txt on your machine.

- Meta guide
- Tool matrix (abridged)
- Program overview
- Voice guide

<div className="my-3 rounded-md border border-fd-border bg-fd-card p-4 text-sm">
  <PersistentCheckbox id="setup-step-4" label="Uploaded all 4 core files to the Project knowledge" />
</div>

## 4. Start your first chat

Open a new chat inside the Project. Paste [Prompt 1, Define your customer](/docs/prompts/define-customer). Fill in the business-idea placeholder. Run it.

<div className="my-3 rounded-md border border-fd-border bg-fd-card p-4 text-sm">
  <PersistentCheckbox id="setup-step-5" label="Ran my first prompt inside the Project" />
</div>
